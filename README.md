# BMI 计算器

一个简单的网页版 BMI 计算器，可以帮助用户计算和了解自己的身体质量指数（BMI）。

## 功能

- 输入身高和体重
- 计算 BMI 值
- 显示体重状态分类

## 未来计划

- 添加科学体重建议功能
- 集成大模型 API
- 考虑更多健康指标（年龄、性别、血压等）

## 使用方法

1. 访问网页
2. 输入您的身高（厘米）
3. 输入您的体重（公斤）
4. 点击"计算 BMI"按钮
5. 查看结果
